About this project:

It is made by Tomasz Pieczkowski. It allows user to solve sudoku by himself with tips in form of placeholder numbers that can be inserted. The idea behind this project was to create web app that would make use of specific technologies and would be useful. And what's exactly useful about this application? Well it is something that my grandparents can use. I hope they like it!

Rules of sudoku:

Each row, column and block (3x3 subgrid) must contain each number from 1 to 9 exactly once.

Usage:

Numbers that are still valid for a given cell are displayed as placeholders. Invalid inputs and moves that break the rules of sudoku are not put into the fields. Before being able to use the solve button the user is asked to finish setup. To do so user has to fill in at least 17 cells in a way that makes sudoku solvable (it is checked after user presses the button). The solve button asks server to calculate solution and displays it. Generate feature is not yet implemented.
